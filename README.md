# ALU
#Descripcion
Programa de una ALU basado en la arquitectura RISC-V, descripción realizada por verilog y pruebas realizadas por Model Sim.
Este proyecto de Servicio Social se realizo con el fin de entender de mejor forma como funciona el lenguaje de Verilog, ademas de entender como es el funcionamiento de esta y como se comporta esta junto a mas elementos de un procesador.

#Contenido
Operaciones basicas de RISC-V: suma, resta, AND, OR, X-OR, desplazamiento a la derecha, desplazamiento a la izquierda, y sus respectivas banderas de desbordamiento de datos y detector de 0.

#Estructura
En el archivo .v se encuentra el codigo de la ALU. 
En el archivo .vt se encuentra el testbench.
Diapositivas con mas informacion y evidencias.

#Herramientoas
Se uso FPGA DE 10 LITE, Quartus version 20.1, Verilog y Model Sim.
